Valorant

Le jeu, dont le développement commence en 2014, est annoncé pour la première fois sous le nom de code Project A en octobre 2019.
Il a d'abord été annoncé pour être publié sur Windows durant l'été 2020, sans aucun portage sur console et macOS confirmé, et sa phase de bêta fermée a débuté le 7 avril 2020.
Il s'agit du premier jeu développé par Riot Games dans le genre, et il est finalement sorti en version finale le 2 juin 2020.
